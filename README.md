# Morning Investment Briefing

An automated daily investment briefing dashboard covering:

- Global equity markets
- FX markets
- Commodities
- Government bond yields
- Economic indicators
- Market news
- Portfolio monitoring

## Objective

The aim of this project is to build a tool that provides a daily morning briefing similar to what an investment analyst would use before starting the day.
